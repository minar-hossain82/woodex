# 🪑 Woodex - Furniture E-Commerce Landing Page

Woodex is a modern and responsive furniture e-commerce landing page designed to showcase premium furniture collections, featured products, blogs, and interior design inspiration. The website delivers an engaging shopping experience with a clean UI and responsive design.

## 🌐 Live Demo

🔗 https://minar-woodex.vercel.app/

## 📂 GitHub Repository

🔗 https://github.com/minar-hossain82/woodex

## ✨ Features

- 🪑 Modern furniture e-commerce landing page
- 📱 Fully responsive design
- 🛋️ Featured products showcase
- ⭐ Popular furniture collection section
- 🏠 Interior design inspiration
- 📰 Latest blog section
- 🛒 Shopping call-to-action sections


### Go to Project Folder

```bash

```

### Open Project

Simply open `index.html` in your browser.

Or use VS Code Live Server.
